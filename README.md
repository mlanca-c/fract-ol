# Cloning

 ```
 git clone --recursive git@github.com:mlanca-c/fract-ol.git
 cd fract-ol
 make all
 ```
 
# Compiling
 
 ```./fractol <fractal> [x] [y] [-p <precision>] [-c <color>] ```
  
  The ```<>``` are mandatory and the ```[]``` are optional
 
 **example:**
 ```Shell
 ./fractol Mandelbrot -p 255 -c 0x0000ff
 ./fractol Julia 1 0 -c 0x00ff0000
 ```

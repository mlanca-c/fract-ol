# Cloning

 ```
 git clone --recursive git@github.com:mlanca-c/fract-ol.git
 cd fract-ol
 make all
 ```
 
# Compiling
 
 ```./fractol <fractal>```
  
  The ```<>``` are mandatory and the ```[]``` are optional
 
 **possibilities:**
 ```Shell
 ./fractol Mandelbrot
 ./fractol Julia
 ```
```
COLOR OPTIONS:
	1: black and white
	2: red
	3: green
	4: blue
```
